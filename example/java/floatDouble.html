Java 浮點數格式及精度

參考資料

IEEE二進位浮點數算術標準（IEEE 754)

浮點小數表示法

$51.25_{(10)}$ = $110011.01_{(2)}$。

$110011.01_{(2)}$ 以浮點小數表示法表示為 $1.1001101 * 2^{5}$。

相當於$1*2^{5}+1*2^{4}+1*2^{1}+1*2^{0}+1*2^{-2}=32+16+2+1+0.25=51.25$。

二進位首碼必定為1，故可省略不記，計算時在補上。

$1.1001101$省略首碼後 => 得到尾數為$1001101$。

$1.1001101 * 2^{5}$ => 得到指數為$5$。

Float 32位元 單精度浮點數

sign(符號)[31]:

0為正，1為負。

exponent(指數)[30~23]:

範圍1~254(0,255為特殊值)。
分正負 以中間值127為0。
故最小值為-126、最大值為127。

fraction(尾數)[22~0]:

二進位首碼必定為1，可省略不記。
故可記錄最大值為$2^{24}$。
空格補0。
精度為$log_{10}(2^{24})=7.2247$ => 7位數。
