Java 浮點數格式及精度

浮點小數表示法

十進位51.25 = 二進位10011.01

二進位10011.01 以浮點小數表示法表示為 1.001101 * 2^5

二進位首碼必定為1，故可省略不記，計算時在補上。

1.001101省略首碼後 => 得到尾數為001101。

1.001101 * 2^5 => 得到指數為5。

Float(4 Byte = 32 bit)32位元單精度

Sign(1) Exponent(8) Fraction(23)

sign(符號):0為正，1為負。

exponent(指數):範圍1~254(0,255為特殊值)，分正負 以127為0，故最小值為-126、最大值為127。

fraction(尾數):二進位首碼必定為1，可省略不記，故可記錄最大值為2^24，精度為log(2^24)=7位數。

Double(8 Byte = 64 bit)64位元雙精度

Sign(1) Exponent(11) Fraction(52)

sign(符號):0為正，1為負。

exponent(指數):範圍1~2046(0,2047為特殊值)，分正負 以1023為0，故最小值為-1022、最大值為1023。

fraction(尾數):二進位首碼必定為1，可省略不記，故可記錄最大值為2^53，精度為log(2^53)=15位數。
